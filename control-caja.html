Control de Caja

⚠️ Importante - Naturaleza de la Caja:

La caja es un saldo virtual de control, no representa dinero físico. Su propósito es tener trazabilidad de cuánto dinero se ha depositado, cuánto se ha debitado y cuánto saldo queda disponible en cada caja (ej: Caja Lima). Este control permite saber en todo momento el estado de los fondos virtuales asignados a cada caja.

Proceso de Débito Automático

¿Cómo funciona el débito automático?

  1. Al calcular viáticos: El sistema asigna automáticamente la caja según las reglas configuradas (base del tripulante o destino del vuelo).
  2. Al aprobar los cálculos: Se genera el débito automático en la caja correspondiente.
  3. Actualización de saldo: El saldo resultante se calcula como: Saldo Inicial + Abonos - Débitos
  4. Notificaciones: Si el saldo queda negativo o se acerca al límite, se genera una alerta automática.
  5. Regularización: Si hay saldo negativo, se puede regularizar posteriormente mediante un abono.

Sobre el Saldo Negativo

✓ Está permitido: El sistema permite que una caja quede con saldo negativo. Esto puede ocurrir cuando los débitos automáticos superan el saldo disponible.

⚠️ Alertas automáticas: Cuando una caja queda con saldo negativo, el sistema:

  • Marca el saldo en rojo
  • Genera una alerta automática visible en el dashboard
  • Envía notificación por email a los responsables
  • Permite regularización posterior mediante abono

🔄 Regularización: Para regularizar un saldo negativo, simplemente registre un abono que cubra el monto negativo más el saldo necesario para operaciones futuras.
