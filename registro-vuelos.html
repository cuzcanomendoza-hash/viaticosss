Importar Reporte desde AIMS

Proceso Automatizado: El sistema se conectará con AIMS para extraer el reporte diario con los vuelos programados, tripulantes, fechas y horarios.
Seleccione la fecha del reporte a importar desde AIMS
Nota: El sistema validará que no existan registros duplicados basándose en: ID Tripulante + Fecha + Vuelo + Rol + Puerto de Pago

Datos que se importarán desde AIMS:

  • ID Tripulante
  • Nombre del Tripulante
  • Rol Cumplido
  • Vuelo / Tramo
  • Fecha de Operación
  • Horarios (Presentación, Salida, Llegada)
  • Base del Tripulante
  • Destino Final
