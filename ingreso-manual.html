Ingreso Manual de Viáticos Especiales

Casos Especiales: Use este módulo para registrar viáticos que no se extraen automáticamente del reporte AIMS, como apoyos a otras bases por más de un mes o comisiones de servicio (CRSV).

Tipo de Ingreso

El sistema buscará automáticamente el nombre desde la API
Clasifique el tipo de movimiento según la reunión: Programado (operación normal), Contingencia (cambios no previstos), Apoyo u Otro
Formatos permitidos: PDF, JPG, PNG (Obligatorio)
Origen del Dato: Este movimiento será identificado como MANUAL para efectos de trazabilidad y auditoría. Todos los movimientos manuales quedan marcados con esta identificación.
Validación Automática: El sistema validará automáticamente que no existan duplicados basándose en ID Tripulante, Fecha y Tipo de Caso.

Carga Masiva de Viáticos Manuales

Formato CSV Requerido: El archivo debe contener las siguientes columnas: ID_Tripulante, Clasificacion, Tipo_Caso, Base, Destino, Fecha_Inicio, Fecha_Fin, Monto, Moneda, Caja, Motivo
Nota: Clasificacion debe ser: programado, contingencia, apoyo u otro. Todos los registros de carga masiva se marcarán como MANUAL.
Tamaño máximo: 10MB
