Configuración de Asociación Automática de Cajas

Importante: Configure cómo el sistema asigna automáticamente la caja al calcular viáticos. El sistema evalúa las reglas en orden de prioridad.
Prioridad Tipo de Regla Criterio Caja Asignada Estado Acciones
1 Por Base del Tripulante Base = SCL SCL Activa
1 Por Base del Tripulante Base = LIM LIM Activa
2 Por Destino del Vuelo Destino = SCL SCL Activa
2 Por Destino del Vuelo Destino = LIM LIM Activa

Lógica de Asignación

  1. El sistema evalúa primero las reglas por Base del Tripulante (Prioridad 1)
  2. Si no encuentra una regla para la base, evalúa las reglas por Destino del Vuelo (Prioridad 2)
  3. Si no encuentra ninguna regla, el sistema generará una alerta para asignación manual
  4. Una vez asignada la caja, el débito se realiza automáticamente al aprobar los cálculos

Nueva Regla de Asociación de Caja

Ejemplo: Si crea una regla "Por Base del Tripulante" con criterio "SCL" y caja "SCL", todos los tripulantes con base en Santiago tendrán sus viáticos debitados automáticamente en la Caja SCL.
